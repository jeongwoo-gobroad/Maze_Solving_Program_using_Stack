Cautions
주의사항

1. 본 소스코드는 최초 스터디용으로 공유되었으며, 대학 과제의 성격도 동시에 띠고 있기 때문에, 학업 과제 제출용 목적으로의
활용을 일체 금함.

2. 학업 과제용 목적이 아닌 경우, 소스코드의 일부 또는 전체를 활용하는 것을 허락함. 단 원 저작자를 표기해야 함.

3. 미로 파일로 최초 입력되어 있는 maze1.txt, maze2.txt, maze3.txt, maze4.txt는 2021년 시점을 기준으로 경북대학교
대구캠퍼스 IT대학 이은주 교수님께 그 저작권이 있음을 명시함. 

4. Windows 10 (64-bit. Intel), Microsoft Visual Studio 2019 버전에서의 작동이 보장되며, 시스템 함수에 의존하는 부분이 있기에 타 플랫폼에서의 정상 동작을 보장하지 않음.
